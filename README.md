# SmartSG
A Cloud-Based Student Housing Decision Platform

## To run it
1. Put below in .env under frontend folder.
```python
#For local run
VITE_API_BASE_URL=http://localhost:8080
```
3. Start backend by
```bash
sudo apt update    
sudo apt install openjdk-17-jdk  
./mvnw spring-boot:run    
```
3. Start frontend by
```bash 
npm install    
npm run dev
```
